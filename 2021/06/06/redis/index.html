redis

redis 初识

Redis 是什么

  • 开源的 nosql
  • 基于健值的存储服务系统
  • 多种数据结构
  • 性能高

数据存储在内存中
C语言实现
单线程架构
6.0版本支持多线程
充分利用服务器 CPU 资源
多线程任务可以分摊 Redis 同步 IO 读写负荷

特性：

  • 速度快
    • 数据村在内存中
    • C 语言
    • 单线程
    • 6.0版本支持多线程
      • 可以充分利用服务器 CPU 资源，目前主线程只能利用一个核。
      • 多线程任务可以分摊 Redis 同步 IO 读写负荷
      • https://www.cnblogs.com/gz666666/p/12901507.html
  • 持久化
    • 数据保存在内存中，对数据的更新将异步地保存到硬盘上（断电不丢数据）
  • 多种数据结构
    • 字符串，hash，列表，集合，有序集合
  • 支持多种编程语言
    • java,PHP,ruby,lua,node
  • 功能丰富
    • 发布订阅（可以实现很多基于消息的功能）
    • Lua脚本（自定义命令）
    • 事务
    • pipeline(提高客户端的并发效率)
  • 简单
    • 23000 行代码、
    • 不依赖外部库
    • 单线程模型
  • 主从复制
  • 高可用、分布式
    • 高可用，redis-sentinel(v2.8)支持高可用
    • 分布式，redis-Cluster(v3.0)支持分布式

典型使用场景

  • 缓存系统
  • 计数器
  • 消息队列系统
  • 排行榜
  • 社交网络
  • 实时系统
  • XXX

redis 可执行文件说明

redis-server -> redis 服务器
redis-cli -> redis 命令行客户端
redis-benchmark -> redis 性能测试
redis-check-aof -> AOF 文件修复工具、
redis-check-dump -> RDB 文件检查工具
redis-sentinel -> Sentinel 服务器
