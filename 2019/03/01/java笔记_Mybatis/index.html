什么是MyBatis

  1. MyBatis 持久层：简化工作量、灵活
  2. Spring 粘合剂：整合框架、AOP、IOC、DI
  3. SpringMVC 表現层：方便前后端数据的传输

MyBatis 是对 jdbc 的封装，将 SQL 语句放在映射文件中（XML），可以自动将输入参数映射到 SQL 语句的动态参数上，自动将 SQL 语句映射成 Java 对象。
