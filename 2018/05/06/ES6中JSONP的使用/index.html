上一篇文章说了JSON、JSONP、AJAX、跨域之间的联系与区别，在结算JSONP的环节中，也深刻的通过举例说明，解释了JSONP实现的原理。但是都是通过简单的例子，今天我们说说我在项目中是怎么使用JSONP的。这里我用到的是github上面一个简单的JSONP接口

API

它的实现源码在index.js中，有兴趣的童鞋可以看看，只有几十行代码。我们先说说他的API:

jsonp(url, opts, fn)

  • url (String) url
  • opts (Object) 可选
    • param (String) 与后台jsonp接口约定好的回调参数名称(默认为 callback)
    • timeout (Number) 超时报错时间(默认为 60000ms),即请求超过1分钟报超时
    • prefix (String) 处理jsonp响应的全局回调函数的前缀（默认为jp）
    • name (String) 处理jsonp响应的全局回调函数的名称（默认为前缀+递增计数器）
  • fn callback 回调函数，带有err和data两个参数
