在上一篇文章JSON、JSONP、AJAX和跨域中，我比较详细的解释了JSONP的原理和简单的应用，包括Jquery在AJAX方法中封装了JSONP。但是这是我们在ES6以前，或者使用Jquery开发时候应用的方法，今天我举例说说在ES6编码环境下我使用的一个例子。

我在项目中用到的也是现成封装好的JSONP方法，大家可以点进去看看，有兴趣的可以看看index.js中的实现方式，原理都是一样的。下面我们就来详细介绍一下使用:

直接使用

$ npm install jsonp 或者直接下载js文件在页面引入。

它的api：

jsonp(url, opts, fn)

  • url (String) url to fetch 拼接好的url,带有与后台约定的参数名称的url（根据实际情况）
  • opts (Object), optional可以选参数
    • param (String) 指定回调的查询字符串参数的名称，即与后台约定的回调方法
    • timeout (Number) 请求超时错误，默认60000ms
    • prefix (String)处理jsonp响应的全局回调函数的前缀（默认为__jp）
    • name (String) 处理jsonp响应的全局回调函数的名称（默认为前缀+递增计数器）
  • fn callback回调 带有两个参数(err,data)

然后根据api传参调用jsonp方法，在callback回调中处理数据，这里就不多说了。
